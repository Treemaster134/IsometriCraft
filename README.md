#Controls:
 - Space - Jump
 - WASD - Move the player
 - Arrow keys - Move the cursor
 - Home - move cursor up
 - End - move cursor down
 - Right Ctrl - switch cursor mode
 - E - place/destroy block

 - Ctrl + S - Save world
 - Ctrl + O - Load world
